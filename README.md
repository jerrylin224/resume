# 五倍紅寶石的履歷（[成品連結](https://afternoon-sea-48996.herokuapp.com/)）

這份作品是為了應徵紅寶石的後端職缺所製作的。

我的目的是想利用Landing page的形式呈現吸引人的履歷，並結合前後端的知識直接打造一個產品

在這份作品學到/完成了:

+ 如何使用Bootstrap和jQuery
+ Codecademy的jQuery課程
+ 利用Grid System處理RWD
+ 利用Sinatra這個套件建立網頁
+ 串接pony和sendgrid
